# **XBook Project**
## A web app that allows users to exchange their books for read, or exchange forever.
### Tools & Technologies : 
- **Main Tools :**
    - Next js (main website).
    - React js (dashboard).
    - Node js & Express (API).
    - MongoDB & mongoose (database)
